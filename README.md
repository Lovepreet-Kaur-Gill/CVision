CVision is a high-performance, full-stack resume builder designed to bridge the gap between candidates and ATS algorithms.

### 🌟 Core Intelligence
- **AI Content Enhancement:** Real-time optimization of bullet points using **Gemini 2.0-Flash**.
- **Smart ATS Mapping:** Automated keyword extraction and alignment with Job Descriptions.
- **Persistent Storage:** Seamless data sync with **Supabase** & **Clerk Auth**.
- **Live Sharing:** Unique public URLs for every resume via dynamic Next.js routing.

### 💻 Technical Stack
- **Frontend:** Next.js 16, Tailwind CSS, Lucide Icons
- **Backend:** Next.js API Routes, Supabase
- **AI/ML:** Google Generative AI (Gemini SDK)
- **Auth:** Clerk (Multi-tenant)
